# Лабораторные работы по предмету "Эффективное программирование современных микропроцессоров и мультипроцессоров"
Каждая из лабораторных работ выполняет задачу решение уравнения Пуассона методом Якоби. В соответствующей папке находится условия лаборатнорной работы и отчёт по ней.

Lab1 - однопоточная программа с базовой оптимизацией посредством компилятора

Lab2 - однопоточная программа с векторизацией вычислений посредством ключей компилятора, директив openmp и intel vector intrinsics

Lab3 - однопоточная программа с оптимизацией памяти при проходе цикла (уменьшение колчества операций загрузки в кэш процессора)

Lab4 - многопоточная программа с оптимизацией по памяти
