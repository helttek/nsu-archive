# Методы трансляции и верификации
Работы по предмету написаны на языке flow9
## Работы
- Реализация функции преобразования алгебраического выражения в отношение двух многочленов (algebraic_expr_to_rational_function)
- Калькулятор с операциями суммирования и умножения (calculator)
- Реализация функции дифференцирования алгебраического выражения по заданной переменной (differentiation)
- Расширенный калькулятор с операциями вычитания, деления, сменой знака и переменными (extended_calculator)
- Обратная польская запись (reverse_polish_notation)
- Реализация функции упрощения рациональной функции по заданному набору тождеств (simple_rational_function)
