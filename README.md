# Архив проектов и учебных заданий во время обучения в НГУ
Наиболее значимые проекты вынесены в отдельные репозитории.

- Вычислительная математика (comp_math)
- Введение в C# и платформу .NET (csharp)
- Базы данных (db)
- Эффективное программирование современных микропроцессоров и мультипроцессоров (epsmim)
- ЭВМ и периферийные устройства (evm)
- Факультатив "Практическая иформационная безопасность" (infosec)
- Программируемые микроконтроллеры (microcontrollers)
- Методы трансляции и компиляции (mtk)
- Объектно-ориентированное программирование (oop)
- Основы параллельного программирования (opp)
- Основы операционных систем (os)
- Основы теории управления (otu)
- Распределенные информационные системы (ris)
- Введение в теорию кодирования (teorkod)
- Управление производственным процессом разработки программного обеспечения (upprpo)
- Сети (networks)
